## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio
- Android SDK
- Git

### Clone Repository

```bash
git clone https://github.com/Art0citus/Expenny.git
```

### Move Into Project Directory

```bash
cd Expenny
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
